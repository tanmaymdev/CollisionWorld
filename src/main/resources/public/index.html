Welcome to the Collision World!!!

  • From the "Type of Ball/Fish" DropDown select the type of Object
  • Post that select the required behavior from the "Ball/Fish Behavior" Dropdown
  • "*" Marked behaviors are not applicable for Fish type
  • You can select the appropriate Behavior as per your liking (It is Random by default)
    1. Vertical,Horizontal and Diagonal --> Straight Line
    2. Rotating --> Object will rotate in circular Path.
    3. Magic --> Supernatural Fish/Ball, can go anywhere (Fish Straight outta Aquaman)
    4. Lazy --> Just like humans Ball/Fish are also Lazy, they stop working after some time
    5. Breathing --> These Ball/Fish breath pretty heavy
    6. Idle --> Who doesn't like sitting idle, if humans can why not Ball/Fish
    7. Colorful --> These Balls change color
    8. Gravity --> Ohh...these are interesting ones, trying to escape the right wall's force of attraction , if their velocity is enough they'll escape otherwise slowly right wall will swallow them
    9. ColorFul and Breathing
    10. Rotating and Breathing
    11. Lazy and Breathing
    12. Gravity and Breathing
  • Then you can select the appropriate Collision Strategy for Balls(It is Random by default)
    1. Color --> These Balls spread colors, as one ball passes over other both change color
    2. Normal --> These Balls don't like each other, they change direction (take opp.)after collision (Kinda like the collision with a wall)
    3. Movement --> After Collision the movement Strategy gets changed for Balls
    4. Velocity --> Increase the Velocity of One and Decrease the Velocity of one, but no direction change
    5. Eating --> These are the dangerous ones, they'll eat the ball which are smaller than them but after eating there size decreases
    6. Normal and Gravity ---> These Balls push the other in the Gravity web of the left wall after the collision
    7. Normal and Breathing --->The Balls which collide start to show the Breathing behavior
    8. Rock --> These are the chill ones, you collide with them nothing will happen. They'll stand the ground just like a rock
    9. Magic --> After Colliding with this Ball, you can end up anywhere in the canvas (I don't know where)
    10. Normal and Color
    11. Normal and Velocity
  • While creating a Ball/Fish you can select the Switchable behavior using the radio button (By default all are non-switchable)
  • Then you Switch the Strategy by clicking on "Switch Strategy" Button
  • While creating a Ball/Fish you can select the Removable behavior using the radio button (By default all are non-removable)
  • Then you remove these set of Ball/Fish by clicking on "Remove Some" Button
Happy Destruction!!
